# NLP Text Classifier

BERT-based multi-class text classifier fine-tuned with LoRA for sentiment analysis, served through a FastAPI endpoint with batched, quantized inference and MLflow experiment tracking.

## Tech Stack

Python, HuggingFace Transformers, PyTorch, PEFT (LoRA), FastAPI, Docker, MLflow

## Dataset

`data/sentiment_dataset.csv` — 600 synthetically generated product review sentences (200 each for positive, negative, neutral), built from templated phrasing across 20 product categories. Generated by `src/generate_dataset.py`, which can be re-run to regenerate or extend the dataset.

## Setup

```bash
pip install -r requirements.txt
```

## Run Pipeline

```bash
python src/generate_dataset.py
python src/data_loader.py
python src/train.py
python src/evaluate.py
python src/quantize.py
```

Training fine-tunes `bert-base-uncased` with a LoRA adapter (rank 8), merges the adapter into the base weights, and saves the result to `models/sentiment-bert-lora`. This requires internet access to download the base model from the HuggingFace Hub the first time it runs.

## Run API Locally

```bash
uvicorn api.main:app --reload
```

`POST /predict` accepts a batch of texts:

```json
{"texts": ["This product is amazing.", "It broke after a day."]}
```

Set `USE_QUANTIZATION=false` to disable dynamic INT8 quantization and serve the full-precision model.

## Run with Docker

```bash
docker build -t nlp-text-classifier .
docker run -p 8000:8000 nlp-text-classifier
```
